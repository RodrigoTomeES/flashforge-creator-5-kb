Remove the packaging bag. Take out the top cover along with the protective foam inside the chamber.

Take out the accessories stored inside the protective foam.

![creator5p\_unboxing\_07.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_07.jpg)

## Step 4: Install the Display Screen [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/unboxing-your-creator-5-pro#step-4-install-the-display-screen)

1.  Align the slots on the back of the display screen with the mounting points on the printer. Then, attach the display screen as shown.  
    ![creator5p\_unboxing\_09.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_09.jpg)
    
2.  Secure the display screen using two M3×6 screws.  
    ![creator5p\_unboxing\_10.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_10.jpg)
    

## Step 5: Install the Extruder Module [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/unboxing-your-creator-5-pro#step-5-install-the-extruder-module)

1.  Insert the four nozzle blockers into the slots of the extruder dock as shown.  
    ![creator5p\_unboxing\_11.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_11.jpg)  
    ![creator5p\_unboxing\_12.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_12.jpg)
    
2.  Align the extruder module with the slot on the right side of the printer and insert it downward. Make sure the filament inlet is facing outward.  
    ![creator5p\_unboxing\_13.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_13.jpg)
    
3.  Secure the extruder dock module using two M3×16 screws.  
    ![creator5p\_unboxing\_14.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_14.jpg)
    
4.  Mount the four extruders onto the extruder dock, aligning each with its corresponding positioning post according to the order of the extruder connection cables.  
    ![creator5p\_unboxing\_15.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_15.jpg)
    
5.  Connect the extruder mount cable to the communication port on the extruder mount. Once firmly inserted, secure it with two ST2.9×9.5 screws.  
    ![creator5p\_unboxing\_16.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_16.jpg)
    

## Step 6: Unlock the Print Bed [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/unboxing-your-creator-5-pro#step-6-unlock-the-print-bed)

Use a 2.5mm Allen wrench to remove the securing screws indicated by the arrows to unlock the print bed.

![creator5p\_unboxing\_17.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_17.jpg)

## Step 7: Install the Top Cover [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/unboxing-your-creator-5-pro#step-7-install-the-top-cover)

1.  Slide each top cover panel into the grooves of the top cover pillars as shown. Make sure the sides with the clips are all facing the same direction during installation.  
    ![creator5p\_unboxing\_18.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_18.jpg)
    
2.  Align the top cover lid with the clips and press down until it snaps into place. Secure the lid using four ST2.9×9.5 screws.  
    ![creator5p\_unboxing\_19.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_19.jpg)  
    ![creator5p\_unboxing\_20.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_20.jpg)  
    ![creator5p\_unboxing\_21.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_21.jpg)
    
3.  Once the top cover is assembled, position it onto the printer by aligning the positioning tabs on the front and rear panels with the corresponding holes on the printer.  
    ![creator5p\_unboxing\_22.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_22.jpg)  
    ![creator5p\_unboxing\_23.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_23.jpg)
    

## Step 8: Install the Spool Holder [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/unboxing-your-creator-5-pro#step-8-install-the-spool-holder)

Remove the spool holder stickers. Insert a spool holder into the mounting hole on the right side of the printer as shown, and secure it with one M3×16 screw.

Repeat this process to install the remaining three spool holders.

(Note: Install the spool holders facing outward to prevent interference with filament feeding.)

![creator5p\_unboxing\_24.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_24.jpg)  
![creator5p\_unboxing\_25.jpg](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/unbox/creator5p_unboxing_25.jpg)

Unboxing is completed.
