## Home [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#home)

![printer-gui-home.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-home.png)

1.  **Home**  
    Displays the current operating status.
    
2.  **Filament Settings**  
    Opens the Filament Settings screen, where you can configure the filament type and color for each spool.
    
3.  **Move & Calibrate**  
    Opens the Move & Calibrate screen. The Move screen allows for manual control of the motion system, while the Calibrate screen lets you select and run specific calibration tasks.
    
4.  **Settings**  
    Opens the Settings menu to configure various printer parameters.
    
5.  **Print File**  
    Displays print files stored in internal storage or a connected USB drive. Select a file to start printing.
    
6.  **Status Bar**  
    Displays the printer name and the following hardware status indicators:
    
    -   ![usb\_disk\_icon.png](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/printer-gui-instructions/usb_disk_icon.png) USB Drive  
        Indicates a USB drive is inserted into the USB port.
    -   ![wi-fi-icon.png](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/printer-gui-instructions/wi-fi-icon.png) Wi-Fi  
        Indicates the printer is connected to a Wi-Fi network.
    -   ![camera-icon.png](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/printer-gui-instructions/camera-icon.png) Camera  
        Indicates the camera is installed and functioning.
7.  **Nozzle Temperature Status**  
    Displays the target and current nozzle temperatures for each extruder. Tap at any time to adjust temperature settings.
    
8.  **Bed Temperature Status**  
    Displays the target and current bed temperatures. Tap at any time to adjust temperature settings.
    
9.  **Chamber Temperature Status**  
    Displays the target and current chamber temperatures. Tap at any time to adjust temperature settings.
    

> Only applicable to Creator 5 Pro.

10.  **Filament Color**  
     Displays the configured filament colors. When the printer is idle, tap for details.
     
11.  **Device Information**  
     Tap when idle to view device details; tap during printing to view print details.
     
12.  **Chamber Light**  
     Toggles the chamber light on or off. The icon is highlighted when the light is on.
     
13.  **Print Mode**  
     Tap to choose from three modes: **Silent**, **Standard**, and **Motion**. The icon updates to reflect the selected mode.  
     For more details, see [Change Print Mode](print-controls.md#change-print-mode).
     

## Filament Settings [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#filament-settings)

![printer-gui-filament-edit.png](https://wiki.flashforge.com/resource/pictures/%E4%B8%AD%E6%96%87/creator5/printer-gui-instructions/printer-gui-filament-edit.png)

1.  **Extruder Selection**  
    Tap to select an extruder.
    
2.  **Edit**  
    Configure the filament type and color for the selected extruder.
    
3.  **Load**  
    Load filament into the selected extruder.
    
4.  **Unload**  
    Displays instructions for manual filament unloading.
    

## Move & Calibrate [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#move-calibrate)

### Move [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#move)

![printer-gui-xyz-control.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-xyz-control.png)

1.  **Position**  
    Displays the current coordinates of the X, Y, and Z axes.
    
2.  **Home**  
    Returns all motion components to their mechanical home positions.
    
3.  **Unlock**  
    Releases the motor locks, allowing you to manually move the extruder mount or the bed.
    
4.  **Horizontal Movement**  
    Moves the extruder along the X and Y axes via the arrow buttons. Tap the center icon to select and load an extruder.
    
5.  **Vertical Movement**  
    Moves the bed along the Z-axis.
    
6.  **Extrusion Control**  
    Extrudes or retracts filament. Ensure the extruder is loaded and the nozzle is heated to the target temperature before use.
    
7.  **Increment**  
    Sets the distance the X, Y, and Z axes move with each tap of an arrow button.  
    For more details, see [Motion Control](XYZ-E-movement-control.md).
    

### Calibrate [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#calibrate)

![printer-gui-xyz-calibration.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-xyz-calibration.png)

Manually perform Extruder Offset calibration, or run automatic calibrations such as bed leveling and vibration compensation.

## Settings [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#settings)

![printer-gui-settings.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-settings.png)

1.  **Account**  
    Displays the account bound to the printer. Tap **Exit** to unbind. If unbound, scan the displayed QR code with Flash Studio Mobile to link the device.
    
2.  **Basic Settings**  
    Contains submenus for printer settings and controls.
    
3.  **Timelapse**  
    Allows you to view timelapse videos stored in the internal storage. It supports single or bulk file deletion, as well as exporting video files to a USB drive.  
    For more details, see [Monitoring & Timelapse](monitoring-and-timelapse.md).
    
4.  **Network**  
    Displays the connected Wi-Fi network. Tap to configure network settings.  
    For more details, see [Network Setup](network-setup.md).
    
5.  **Firmware**  
    Displays the current firmware version. Tap to check for updates.  
    For more details, see [Firmware Update](firmware-upgrade.md).
    
6.  **Extruder Settings**  
    Sets the nozzle diameter for each extruder. Update this setting whenever you change the nozzle size to ensure print accuracy.
    
7.  **Guides & Support**  
    Displays a QR code for the complete User Guide, customer support contact info, and the docking status of each extruder.
    

### Basic Settings [\[wiki §\]](https://wiki.flashforge.com/en/creator-series/creator-5-series/manual/printer-gui-instructions#basic-settings)

Tap the up/down arrows on the right side to view more options.

![printer-gui-basic-settings-01.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-basic-settings-01.png)

1.  **Device Info**  
    Displays basic device information, including the device model, total print time, total filament usage, IP address, MAC address, and serial number.
    
2.  **Device Name**  
    Changes the printer display name, which appears in the status bar for quick identification.
    
3.  **Device Location**  
    Changes the printer's location description. This information syncs with the Flash Studio device list for quick identification across different workspaces.
    
4.  **Free Memory**  
    Displays the current available capacity and the total storage capacity of the printer's internal storage.
    

![printer-gui-basic-settings-02.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-basic-settings-02.png)

5.  **Sound**  
    When enabled, the printer chimes at startup and print completion.
    
6.  **Clog Detection**  
    Enables nozzle clog detection. The printer will pause printing and send an alert if a clog occurs.
    
7.  **Language**  
    Changes the default display language of the user interface.
    
8.  **Auto Sleep**  
    Sets the idle time before the printer automatically enters sleep mode.
    

![printer-gui-basic-settings-03.png](https://wiki.flashforge.com/resource/pictures/creator5_en/printer-gui-instructions/printer-gui-basic-settings-03.png)

9.  **Copy Logs**  
    Exports device logs to a USB flash drive. This helps our support team diagnose issues and identify possible causes of failures.
    
10.  **Factory Reset**  
     Restores the printer to its factory default settings.
